# kmword 点击https://unplage.github.io/kmword/
离线背单词；单词发声；纯英文解释；
导入英文语料定制单词库；支持多个单词库的编辑；
除了导入英文小说，还可从https://github.com/kajweb/dict下载单词库，使用extract_txt生成txt文件导入程序中学习；
生词本功能暂不可用；
